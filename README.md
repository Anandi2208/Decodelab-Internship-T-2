# 🔐 Basic Encryption & Decryption System

## 📌 Project Overview

In today's digital world, protecting sensitive information is one of the most important aspects of cybersecurity. Encryption is a technique used to convert readable information into an unreadable format, ensuring that only authorized users can access the original data.

This project demonstrates the basic concepts of cryptography by implementing a simple Encryption and Decryption system using the Caesar Cipher algorithm. The application allows users to encrypt plain text into coded text and decrypt it back to its original form.

The project helps beginners understand how data confidentiality works and introduces fundamental cybersecurity principles.

# ⚙️ Working Principle

The project uses the Caesar Cipher algorithm.

A Caesar Cipher shifts each letter in the message by a fixed number of positions in the alphabet.

Example:

Original Text:

HELLO

Shift Value:

3

Encrypted Text:

KHOOR

Decrypted Text:

HELLO

---

# 🔄 Workflow

### Step 1

User enters a text message.

### Step 2

The system accepts a shift key.

### Step 3

The encryption algorithm shifts each character.

### Step 4

Encrypted text is generated.

### Step 5

The decryption algorithm reverses the process.

### Step 6

Original text is recovered successfully.

---

# 📊 Algorithm

## Encryption

1. Read user input text.
2. Read shift value.
3. For each character:

   * Shift alphabet by given value.
   * Preserve non-alphabetic characters.
4. Generate encrypted output.
5. Display result.

## Decryption

1. Read encrypted text.
2. Apply reverse shift.
3. Restore original message.
4. Display decrypted text.
